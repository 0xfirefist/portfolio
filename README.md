# Three.js Portfolio

A 3D portfolio project built with Three.js and Vite.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features
- Responsive 3D scene
- Rotating cube with phong material
- Dynamic lighting
- Automatic window resize handling
